# Operations — Order Fulfilment Process

## When a Customer Orders

1. You receive a Stripe payment notification (email + dashboard)
2. Check the payment details for:
   - Customer name and shipping address
   - Custom field: "Cap Colour" (their selection)
3. Go to KidsCultureDesign on Etsy: https://www.etsy.com/listing/1433263187/
4. Order with these details:
   - Hat Colour: [whatever the customer chose]
   - Font: Nineties Font
   - Personalization: `single until series b`
   - Shipping address: customer's address (from Stripe)
5. Pay the Etsy seller (£15 UK / £24 intl)
6. Save the Etsy order confirmation
7. When Etsy shows tracking, forward it to the customer via email

## Handling Issues

**Customer asks for a colour not on the site:**
- KidsCultureDesign has 15 colours. Check their listing.
- If available, order it. If not, email the customer.

**Customer wants to return:**
- 14-day window from delivery (UK Consumer Contracts Regulations 2013)
- Quality issues: you cover return postage
- Changed mind: customer pays return postage
- Refund via Stripe once you confirm the return

**Etsy order delayed:**
- Contact KidsCultureDesign via Etsy messages
- Email the customer with an update
- Do not make promises about timelines you can't control

**Supplier discount:**
- Code COMEBACK gives 5% off at KidsCultureDesign
- Use it on every order — it's free margin
